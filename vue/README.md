# Install and Setup

## Install node.js and npm

### Install node.js

[Document for install node.js](https://nodejs.org/en/download/package-manager)

### Install npm

```bash
curl http://npmjs.org/install.sh | sh
```

## Install gulp

```bash
$ npm install -g gulp
```

## Install bower

```bash
$ npm install -g bower
```

# Setup

## Install dependency package

```bash
$ npm install
$ bower install
```

## Run gulp

```bash
$ gulp
```

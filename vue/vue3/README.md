# Vue 3

## 1. 使用 Vite

### 1.1. 创建项目 (交互式)

使用 NPM

```bash
npm create vite@latest
```

使用 Yarn

```bash
yarn create vite
```

使用 PNPM

```bash
pnpm create vite
```

### 1.2. 创建项目 (参数式)

NPM 6.x

```bash
npm create vite@latest my-vue-app --template vue
```

NPM 7.x, 注意需要增加额外的 `--`

```bash
npm create vite@latest my-vue-app -- --template vue
```

Yarn

```bash
yarn create vite my-vue-app --template vue
```

PNPM

```bash
pnpm create vite my-vue-app --template vue
```
